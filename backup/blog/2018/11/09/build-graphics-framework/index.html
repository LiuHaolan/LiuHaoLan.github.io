build graphics framework

When doing graphics research, you need to implement some basic idea and using the another part of the system
as a blackbox. Then you’d better try to stack on your formal work and build a united framework to do this
kind of thing, like a real-time rendering framework, a offline rendering system, etc.
Then the management will be easier and for you to understand your code.
Also, it’s good programming practice, like writing such extendable code.
